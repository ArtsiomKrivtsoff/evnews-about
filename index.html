О нас

Что мы делаем

EV World News — медиапроект, специализирующийся на новостях рынка электромобилей. Мы агрегируем, обрабатываем и публикуем контент из международных источников на русском языке.

Проект использует автоматизированный технологический стек для сбора новостей, их обработки с помощью искусственного интеллекта и публикации в социальных сетях — Telegram и Instagram.

Мы предоставляем технические услуги по автоматизации публикации контента через API социальных платформ, включая Instagram Graph API.

Как работает проект

01
Агрегация новостей
Автоматический сбор новостей об электромобилях из международных RSS-источников через систему автоматизации n8n.
02
Обработка ИИ
Каждая новость обрабатывается языковыми моделями: перевод, редактирование, оценка значимости и форматирование для публикации.
03
Автопубликация
Готовый контент автоматически публикуется через официальные API платформ, включая Instagram Graph API и Telegram Bot API.
